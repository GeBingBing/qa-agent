# 差旅与日常报销标准

## 差旅原则

- 差旅出行前必须在 OA 系统提交差旅申请，含目的地 / 时间 / 预算
- 国内差旅至少**提前 3 天**申请；**国际差旅必须提前 7 天**申请，含签证 / 健康保险手续
- 差旅期间产生的所有费用以本政策为准

## 国内差旅

### 交通

- 经济舱机票 / 高铁二等座 / 普通火车软卧
- 一线 / 新一线城市内市内交通：公交 / 地铁实报实销，打车单次不超过 100 元
- 自驾里程补贴：1.2 元 / 公里

### 酒店

- **国内酒店上限：600 元 / 晚**（含早餐）
- 一线城市（北京 / 上海 / 广州 / 深圳）可上浮至 800 元 / 晚
- 与同级同事同行，建议合住降低成本

### 餐饮

- 标准日均 **400 元**，含早 / 中 / 晚三餐
- 一线城市可上浮至 500 元 / 日
- 单次商务宴请需经直属经理事前审批

## 国际差旅

- 经济舱机票（飞行 8 小时及以上可申请商务舱）
- 国际酒店上限：1500 元 / 晚（折人民币）
- 餐饮：800 元 / 日（折人民币）
- 因公护照 / 签证费用全额报销

## 团队聚餐

- 部门聚餐：人均上限 **200 元**（含晚餐 / 团建）
- 客户宴请：人均上限 600 元，需经部门负责人审批
- 上限以外金额由发起人个人承担

## 报销提交

1. 出差结束 5 个工作日内提交报销单
2. 需附原始发票、行程单、审批通过的差旅申请编号
3. 发票抬头必须为公司全名 + 税号
4. 单笔超过 5000 元附银行流水

## 不予报销

- 个人消费（酒水 / 娱乐 / 礼品）
- 罚款 / 违章 / 超速
- 未提前申请、事后补单的差旅
- 跨年度报销（次年 1 月 31 日截止）
